# 项目启动说明

## 问题分析
你之前遇到的 "501 - Server does not support this operation" 错误是因为有一个简单的 HTTP 服务器在运行，它只支持 GET 请求，不支持 POST 等其他方法。

## 正确启动方式

### 步骤 1：确保没有其他服务器在运行

确保 8000 端口没有被占用。如果有，先停止它。

### 步骤 2：启动 FastAPI 后端（这是正确的服务器）

打开命令行（Command Prompt 或 PowerShell），然后：

```cmd
cd d:\workspace\trae-space\DZDP\AI-dzdp
pip install -r requirements.txt  # 只需要运行一次
uvicorn backend.main:app --reload
```

你应该会看到类似这样的输出：
```
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
INFO:     Started reloader process [XXXXX]
INFO:     Started server process [XXXXX]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

### 步骤 3：打开浏览器

在浏览器中访问：http://localhost:8000
